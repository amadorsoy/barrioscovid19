# Barrios Covid-19

## Explicación

**Barrios Covid-19** solo intenta poner en contacto a las personas de un barrio, una calle o de una población (tanto como se quiera/pueda), la idea es que por un lado alguien inicie un _barrio_ en la aplicación, realmente esto no es más que para crear la **zona geográfica**, a partir de este momento se podrán dar de alta tres tipos de perfiles (por supuesto esta es la idea inicial)

- Perfiles de usuario (ya se buscará el nombre adecuado a cada perfil):
  - Persona necesitado de ayuda: esto equivale a una persona que no tiene forma de ir a realizar la compra.
  - Persona que ayuda: personas que se presentan voluntarias para ir a comprar o a ayudar a las que necesitan ayuda.
  - Tiendas: son las tiendas del barrio que quieren ofrecer sus productos en la aplicación, esto no es un e-commerce, la aplicación se limitará a permitir configurar una lista de productos que se quiere comprar. La tienda se encarga de indicar si puede servir o no el producto. Además la tienda indicará si realiza el servicio a domicilio. También se permitirá que las personas que ayudan puedan asignarse listas de la compra para llevar al domicilio.

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

### Installation

```bash
$ npm install
```

### Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

### Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
